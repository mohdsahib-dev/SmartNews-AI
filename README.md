# SmartNews-AI 📰

SmartNews-AI is an AI-powered news aggregation platform that collects news from multiple sources and provides concise summaries for improved readability and user experience.

## 🌟 Features

* Real-time news aggregation
* AI-powered news summarization
* Category-based news browsing
* Responsive user interface
* Fast and user-friendly experience
* Modern web application architecture

## 🛠️ Tech Stack

* Next.js
* React.js
* JavaScript
* Tailwind CSS
* News API
* AI Summary API
* Vercel

## 💻 Installation

```bash
git clone https://github.com/mohdsahib-dev/SmartNews-AI.git

cd SmartNews-AI

npm install

npm run dev
```

## 🎯 Project Goal

To provide users with quick and meaningful news consumption through AI-generated summaries and categorized content.
